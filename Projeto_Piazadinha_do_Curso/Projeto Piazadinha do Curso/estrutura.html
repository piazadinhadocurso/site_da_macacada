Estrutura básica

A estrutura básica do HTML é definida pelas seguintes tags:

  • DOCTYPE html: essa tag serve basicamente para indicar a versão do HTML que está sendo usada. No caso, a versão é o HTML5.
  • html: é a tag que delimita o documento, ou seja, tudo o que fizer parte da sua página deve estar dentro dessa tag.
  • head: a tag head é o cabeçalho do site, ou seja, onde você definirá todas as configuações e comportamentos do site, como por exemplo os estilos, scripts, título, protocolo UTF-8 e entre outros. Lembrando que o que está dentro dessa tag não ficará visível do conteúdo do site.
  • meta: as tagas meta servem para atribuir algumas configurações e outros fins para o seu documento. Por exemplo, o meta charset="UTF-8" é um identificador de caracteres especiais/acentos para o site.
  • title: o title é o título do site que irá aparecer na aba do seu navegador.
  • body: esse é o local onde irá aparecer todo o seu conteúdo visual do seu site, como os títulos, parágrafos, botões, menus etc.
                
                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                    
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">>
                        <title>Document</title>
                    
                    </head>
                    <body>
                    
                    </body>
                    </html>
