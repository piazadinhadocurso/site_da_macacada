QUAIS SÃO OS CONTEÚDOS DO SITE?

HTML

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Utilizando o CSS é possível alterar cores, fontes, espaçamentos etc...

EXTRA

Além dos dois conteúdos anteriores, esse site comtempla uma série de materiais extras feitos por outros autores e organizações. Dessa forma, o estudo será mais expandido, e em consequência, um melhor aprendizado a todos.
